## How to use ?

Inside the folder make sure you have `tasks.txt` and `tasks_completed.txt`

### In a terminal

#### For the graphic version install Tkinter
```
pip install tk
```
#### And use :
```
python3 main_gui.py
```

###

#### For the terminal version use :
```
python3 main.py
```
